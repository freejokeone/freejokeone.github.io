Javascript数据结构与算法（概述）

作为一名在大学以设计为主前端为辅的大学生，没有接触专业的计算机理论知识，对计算机数据结构算法编译原理尚未有过专业的训练，反复3遍学习javascript数据结构和算法之后越发感受到掌握数据结构的必要性。有部分人会认为没有计算机专业的计算机理论知识也能完成日常的工作，但在项目中处理复杂的数据处理问题。就越发感受到数据结构的重要性。

正如《数据结构与算法Javascript》所描述的那样：“前端工程师如果自身不重视算法 和数据结构这样的基础知识，很可能陷入数年从事单一重复劳动毫无成长这样的职业发展 困境。在移动浪潮到来之后，用户体验要求越来越高，对前端提出了更高的要求，前端这 个职能，必须提高自身才能继续发展，未来的网页 UI，绝对不是靠几个选择器操作加超链 接就能应付的。越来越复杂的产品和基础库，需要坚实的数据结构与算法基础才能驾驭”

在后面的章节，将依据《Javascripts数据结构与算法》为主要路径，谈谈自己在学完这些内容后的一些看法：

描述的内容包括

  • javascript 常见数据类型
  • 简单算法分析
  • 数组
  • 列表，栈，队列
  • 数
  • 散列
  • 图
  • 排序算法
  • 算法设计
